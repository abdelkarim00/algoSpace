Bucket Sort Algorithm

📘 Overview

Bucket Sort is a distribution sort that distributes elements into a number of buckets, sorts each bucket individually (often using another algorithm like insertion sort), and then concatenates all sorted buckets.

🧠 Algorithm Steps

  1. Find the range of the input data.
  2. Divide the range into a fixed number of buckets.
  3. Distribute the input elements into these buckets.
  4. Sort each bucket individually.
  5. Concatenate all sorted buckets.

⏱️ Time Complexity

  • Best Case: O(n + k)
  • Average Case: O(n + k)
  • Worst Case: O(n²) — when all elements fall into a single bucket and a poor inner sort is used

💾 Space Complexity

  • Auxiliary Space: O(n + k)

✅ Easy Problems Solvable by Bucket Sort

1. Sort Floating-Point Numbers in [0, 1)


void bucketSort(float arr[], int n) {
    std::vector<std::vector<float>> buckets(n);

    for (int i = 0; i < n; i++) {
        int index = n * arr[i];
        buckets[index].push_back(arr[i]);
    }

    for (int i = 0; i < n; i++)
        std::sort(buckets[i].begin(), buckets[i].end());

    int idx = 0;
    for (int i = 0; i < n; i++) {
        for (float val : buckets[i])
            arr[idx++] = val;
    }
}

2. Sort Uniformly Distributed Integers


void bucketSortInt(int arr[], int n, int bucketSize = 10) {
    int maxVal = *std::max_element(arr, arr + n);
    std::vector<std::vector<int>> buckets((maxVal / bucketSize) + 1);

    for (int i = 0; i < n; i++) {
        int index = arr[i] / bucketSize;
        buckets[index].push_back(arr[i]);
    }

    int idx = 0;
    for (auto& bucket : buckets) {
        std::sort(bucket.begin(), bucket.end());
        for (int val : bucket)
            arr[idx++] = val;
    }
}

🔁 Intermediate Problems Solvable by Bucket Sort

1. Sort Student GPA Scores (0.00 to 4.00)


void sortGPA(float gpas[], int n) {
    bucketSort(gpas, n);
}

2. Sort Decimal Numbers in [0, 100) with 2 Decimal Places


void sortDecimals(float arr[], int n) {
    for (int i = 0; i < n; i++)
        arr[i] /= 100.0f;
    bucketSort(arr, n);
    for (int i = 0; i < n; i++)
        arr[i] *= 100.0f;
}

🧠 Hard Problem Solvable by Bucket Sort

1. Sort 2D Points by X Coordinate Using Buckets


struct Point {
    float x, y;
};

void bucketSortPoints(std::vector<Point>& points) {
    int n = points.size();
    std::vector<std::vector<Point>> buckets(n);

    for (auto& pt : points) {
        int index = n * pt.x;
        buckets[index].push_back(pt);
    }

    for (auto& bucket : buckets)
        std::sort(bucket.begin(), bucket.end(), [](Point a, Point b) {
            return a.x < b.x;
        });

    int idx = 0;
    for (auto& bucket : buckets)
        for (auto pt : bucket)
            points[idx++] = pt;
}

📌 Summary

  • Bucket Sort is efficient when input is uniformly distributed.
  • It is useful for floating-point numbers or data falling in a known range.
  • Its time complexity can degrade if data is not well distributed.

❗ When to Use

  • When input is evenly spread over a range (e.g., [0, 1)).
  • For floating point sorting, GPA, percentages, coordinates, etc.
