GCD and LCM Algorithms

üìò Overview

The GCD (Greatest Common Divisor) of two integers is the largest integer that divides both numbers without leaving a remainder. The LCM (Least Common Multiple) is the smallest integer that is divisible by both numbers.

These concepts are fundamental in number theory and have applications in simplifying fractions, cryptography, and solving Diophantine equations.

üîç How It Works

  • GCD: The Euclidean algorithm finds the GCD by repeatedly applying the formula: gcd(a, b) = gcd(b, a % b), until b = 0.
  • LCM: The LCM can be computed using the formula: lcm(a, b) = (a * b) / gcd(a, b).

‚è±Ô∏è Time Complexity

  • GCD (Euclidean Algorithm): O(log(min(a, b))) ‚Äî The algorithm reduces the problem size exponentially with each step.
  • LCM: O(log(min(a, b))) ‚Äî Calculating the LCM involves computing the GCD first.

üíæ Space Complexity

  • Both GCD and LCM: O(1) ‚Äî Only a constant amount of extra space is required.

‚úÖ Easy Problems Solvable by GCD and LCM

1. Finding the GCD and LCM of Two Numbers

Given two integers, compute their GCD and LCM.


#include <iostream>

int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

int lcm(int a, int b) {
    return (a * b) / gcd(a, b);
}

int main() {
    int a = 12, b = 18;
    std::cout << "GCD: " << gcd(a, b) << ", LCM: " << lcm(a, b) << std::endl;
    return 0;
}

üîÅ Intermediate Problems Solvable by GCD and LCM

1. Finding the LCM of Multiple Numbers

Given an array of integers, compute their LCM.


#include <iostream>
#include <vector>

int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

int lcm(int a, int b) {
    return (a * b) / gcd(a, b);
}

int findLcmOfArray(const std::vector<int>& arr) {
    int result = arr[0];
    for (int i = 1; i < arr.size(); ++i) {
        result = lcm(result, arr[i]);
    }
    return result;
}

int main() {
    std::vector<int> numbers = {4, 5, 6};
    std::cout << "LCM of array: " << findLcmOfArray(numbers) << std::endl;
    return 0;
}

2. Solving Diophantine Equations

Given integers a, b, and c, determine if the equation ax + by = c has integer solutions.


#include <iostream>

int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

bool hasSolution(int a, int b, int c) {
    return c % gcd(a, b) == 0;
}

int main() {
    int a = 15, b = 25, c = 100;
    if (hasSolution(a, b, c))
        std::cout << "Equation has integer solutions." << std::endl;
    else
        std::cout << "Equation has no integer solutions." << std::endl;
    return 0;
}

üß† Advanced Problem Solvable by GCD and LCM

1. Finding the Least Common Denominator (LCD) of Fractions

Given a list of fractions, compute their least common denominator.


#include <iostream>
#include <vector>

int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

int lcm(int a, int b) {
    return (a * b) / gcd(a, b);
}

int findLcdOfFractions(const std::vector<std::pair<int, int>>& fractions) {
    int denominator = fractions[0].second;
    for (const auto& fraction : fractions) {
        denominator = lcm(denominator, fraction.second);
    }
    return denominator;
}

int main() {
    std::vector<std::pair<int, int>> fractions = {{1, 2}, {1, 3}, {1, 4}};
    std::cout << "LCD of fractions: " << findLcdOfFractions(fractions) << std::endl;
    return 0;
}
