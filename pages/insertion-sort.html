Insertion Sort Algorithm

📌 Overview

Insertion Sort is a simple sorting algorithm that builds the sorted list one element at a time. It is efficient for small datasets and partially sorted data.

🧠 Algorithm Steps

  1. Start from the second element (index 1).
  2. Compare it with elements before it and insert it into the correct position.
  3. Repeat for all remaining elements.

⏱️ Time Complexity

  • Best Case: O(n) — when the array is already sorted
  • Average Case: O(n²)
  • Worst Case: O(n²) — when the array is sorted in reverse

💾 Space Complexity

  • Space: O(1) — in-place sorting

✅ Easy Problems Solvable by Insertion Sort

1. Sort Small Integer Array


void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

2. Insert Element into a Sorted Array


// Insert 'val' into sorted array 'arr' of size n
void insertIntoSortedArray(int arr[], int &n, int val) {
    int i = n - 1;
    while (i >= 0 && arr[i] > val) {
        arr[i + 1] = arr[i];
        i--;
    }
    arr[i + 1] = val;
    n++;
}

🧠 Hard Problem Solvable by Insertion Sort

1. Real-time Sorting of Streaming Data (e.g. in Sensor Devices)

Insertion Sort works well when elements are coming in one at a time and must be sorted immediately (e.g. microcontroller).


void insertSensorReading(int sorted[], int &n, int newReading) {
    int i = n - 1;
    while (i >= 0 && sorted[i] > newReading) {
        sorted[i + 1] = sorted[i];
        i--;
    }
    sorted[i + 1] = newReading;
    n++;
}
