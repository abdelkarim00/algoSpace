Linear Search Algorithm

📘 Overview

Linear Search, also known as Sequential Search, is a simple algorithm that checks each element of a list sequentially until the desired element is found or the list is exhausted.

🔍 How It Works

  1. Start from the first element of the list.
  2. Compare the current element with the target value.
  3. If they match, return the index of the current element.
  4. If they don't match, move to the next element and repeat the process.
  5. If the end of the list is reached without finding the target, return -1 to indicate the element is not present.

⏱️ Time Complexity

  • Best Case: O(1) — when the target element is the first element in the list.
  • Average Case: O(n) — when the target element is somewhere in the middle.
  • Worst Case: O(n) — when the target element is the last element or not present at all.

💾 Space Complexity

  • Space Complexity: O(1) — since it only requires a constant amount of extra space for variables.

✅ Easy Problems Solvable by Linear Search

1. Find the Index of an Element in an Array

Given an array of integers, find the index of a specific element.


#include <iostream>
using namespace std;

int linearSearch(int arr[], int size, int target) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) {
            return i;
        }
    }
    return -1;
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int size = sizeof(arr) / sizeof(arr[0]);
    int target = 30;
    int result = linearSearch(arr, size, target);
    if (result != -1) {
        cout << "Element found at index: " << result << endl;
    } else {
        cout << "Element not found" << endl;
    }
    return 0;
}

2. Check if an Element Exists in an Array

Given an array of integers, check if a specific element exists.


#include <iostream>
using namespace std;

bool linearSearch(int arr[], int size, int target) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) {
            return true;
        }
    }
    return false;
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int size = sizeof(arr) / sizeof(arr[0]);
    int target = 60;
    if (linearSearch(arr, size, target)) {
        cout << "Element found" << endl;
    } else {
        cout << "Element not found" << endl;
    }
    return 0;
}

🔁 Intermediate Problems Solvable by Linear Search

1. Count the Occurrences of an Element in an Array

Given an array of integers, count how many times a specific element appears.


#include <iostream>
using namespace std;

int countOccurrences(int arr[], int size, int target) {
    int count = 0;
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) {
            count++;
        }
    }
    return count;
}

int main() {
    int arr[] = {10, 20, 30, 20, 40, 20};
    int size = sizeof(arr) / sizeof(arr[0]);
    int target = 20;
    int result = countOccurrences(arr, size, target);
    cout << "Element " << target << " occurs " << result << " times." << endl;
    return 0;
}

2. Find the Maximum Element in an Array

Given an array of integers, find the maximum element.


#include <iostream>
using namespace std;

int findMax(int arr[], int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int size = sizeof(arr) / sizeof(arr[0]);
    int result = findMax(arr, size);
    cout << "Maximum element is: " << result << endl;
    return 0;
}

🧠 Advanced Problem Solvable by Linear Search

1. Find the First Non-Repeating Character in a String

Given a string, find the first character that does not repeat.


#include <iostream>
#include <unordered_map>
#include <string>
using namespace std;

char firstNonRepeatingChar(string str) {
    unordered_map<char, int> freq;
    for (char c : str) {
        freq[c]++;
    }
    for (char c : str) {
        if (freq[c] == 1) {
            return c;
        }
    }
    return '\0'; // No non-repeating character
}

int main() {
    string str = "swiss";
    char result = firstNonRepeatingChar(str);
    if (result != '\0') {
        cout << "First non-repeating character is: " << result << endl;
    } else {
        cout << "No non-repeating character found" << endl;
    }
    return 0;
}

📌 Summary

  • Linear Search is a straightforward algorithm ideal for small datasets or unsorted collections.
  • It is easy to implement but becomes inefficient for large datasets due to its O(n) time complexity.
  • Despite its simplicity, it remains a fundamental algorithm in computer science education and practical applications where data is small or unsorted.
