Modular Arithmetic

Modular arithmetic is a system of arithmetic for integers, where numbers "wrap around" upon reaching a certain value, the modulus. It's widely used in number theory and cryptography.

Algorithms and Operations

Some basic modular operations include:

  • Modular Addition: (a + b) mod m
  • Modular Subtraction: (a - b) mod m
  • Modular Multiplication: (a * b) mod m
  • Modular Exponentiation: (a^b) mod m

Time and Space Complexity

Modular arithmetic operations have the following time and space complexities:

  • Time Complexity:
    • Addition, Subtraction, Multiplication: O(1)
    • Modular Exponentiation: O(log b) using exponentiation by squaring
  • Space Complexity: O(1) for basic operations, O(log b) for storing intermediate results in modular exponentiation

Easy Problems Solvable by Modular Arithmetic

Problem 1: Compute (a + b) mod m

Given integers a, b, and m, compute (a + b) mod m.

    
#include <iostream>
using namespace std;

int modAdd(int a, int b, int m) {
    return (a + b) % m;
}

int main() {
    int a = 5, b = 7, m = 12;
    cout << "Result: " << modAdd(a, b, m) << endl;
    return 0;
}

Intermediate Problems Solvable by Modular Arithmetic

Problem 1: Compute (a^b) mod m

Given integers a, b, and m, compute (a^b) mod m efficiently using exponentiation by squaring.

    
#include <iostream>
using namespace std;

int modExponentiate(int a, int b, int m) {
    int result = 1;
    a = a % m;
    while (b > 0) {
        if (b % 2 == 1)
            result = (result * a) % m;
        b = b >> 1;
        a = (a * a) % m;
    }
    return result;
}

int main() {
    int a = 5, b = 3, m = 13;
    cout << "Result: " << modExponentiate(a, b, m) << endl;
    return 0;
}

Problem 2: Compute Modular Inverse

Given integers a and m, compute the modular inverse of a modulo m. Use the Extended Euclidean Algorithm to compute it.

    
#include <iostream>
using namespace std;

int gcdExtended(int a, int b, int &x, int &y) {
    if (a == 0) {
        x = 0;
        y = 1;
        return b;
    }
    int x1, y1;
    int gcd = gcdExtended(b % a, a, x1, y1);
    x = y1 - (b / a) * x1;
    y = x1;
    return gcd;
}

int modInverse(int a, int m) {
    int x, y;
    int g = gcdExtended(a, m, x, y);
    if (g != 1)
        return -1; // Inverse doesn't exist
    else {
        int res = (x % m + m) % m;
        return res;
    }
}

int main() {
    int a = 3, m = 11;
    cout << "Modular Inverse: " << modInverse(a, m) << endl;
    return 0;
}

Advanced Problem Solvable by Modular Arithmetic

Problem 1: Solve a System of Modular Equations

Given two congruences:

  • x ≡ 2 (mod 5)
  • x ≡ 3 (mod 7)

Find the solution using the Chinese Remainder Theorem.

    
#include <iostream>
using namespace std;

int modInverse(int a, int m) {
    a = a % m;
    for (int x = 1; x < m; x++)
        if ((a * x) % m == 1)
            return x;
    return -1;
}

int solveModularEquations(int a1, int m1, int a2, int m2) {
    int m = m1 * m2;
    int M1 = m / m1;
    int M2 = m / m2;
    int inv1 = modInverse(M1, m1);
    int inv2 = modInverse(M2, m2);
    if (inv1 == -1 || inv2 == -1)
        return -1; // No solution
    int x = (a1 * M1 * inv1 + a2 * M2 * inv2) % m;
    return x;
}

int main() {
    int a1 = 2, m1 = 5, a2 = 3, m2 = 7;
    cout << "Solution: " << solveModularEquations(a1, m1, a2, m2) << endl;
    return 0;
}
