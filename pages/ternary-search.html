Ternary Search Algorithm

ğŸ“˜ Overview

Ternary Search is a divide-and-conquer algorithm that finds the position of a target value within a sorted array. It works by dividing the array into three parts using two midpoints and recursively narrowing down the search range.

ğŸ” How It Works

  1. Divide the array into three parts by calculating two midpoints: mid1 = low + (high - low) / 3 and mid2 = high - (high - low) / 3.
  2. Compare the target value with the elements at mid1 and mid2.
  3. If the target equals arr[mid1], return mid1; if it equals arr[mid2], return mid2.
  4. If the target is less than arr[mid1], recursively search the left third of the array.
  5. If the target is greater than arr[mid2], recursively search the right third of the array.
  6. If the target is between arr[mid1] and arr[mid2], recursively search the middle third of the array.

â±ï¸ Time Complexity

  • Best Case: O(1) â€” when the target element is found at one of the midpoints.
  • Average and Worst Case: O(logâ‚ƒ n) â€” the search space is divided into three parts at each step, reducing the problem size exponentially.

Ternary Search is more efficient than linear search but less commonly used than binary search due to its higher constant factors and more comparisons per iteration.

ğŸ’¾ Space Complexity

  • Iterative Implementation: O(1) â€” uses a constant amount of extra space.
  • Recursive Implementation: O(logâ‚ƒ n) â€” due to the depth of the recursion stack.

1. Finding an Element in a Sorted Array

Given a sorted array of integers and a target value, determine if the target exists in the array.


#include <iostream>
#include <vector>

int ternarySearch(const std::vector<int>& arr, int target) {
    int low = 0, high = arr.size() - 1;
    while (low <= high) {
        int mid1 = low + (high - low) / 3;
        int mid2 = high - (high - low) / 3;
        if (arr[mid1] == target) return mid1;
        if (arr[mid2] == target) return mid2;
        if (target < arr[mid1]) high = mid1 - 1;
        else if (target > arr[mid2]) low = mid2 + 1;
        else {
            low = mid1 + 1;
            high = mid2 - 1;
        }
    }
    return -1;
}

int main() {
    std::vector<int> arr = {1, 3, 5, 7, 9, 11};
    int target = 7;
    int result = ternarySearch(arr, target);
    if (result != -1) std::cout << "Element found at index " << result << std::endl;
    else std::cout << "Element not found" << std::endl;
    return 0;
}

2. Finding the First Occurrence of an Element

Given a sorted array with duplicate elements, find the index of the first occurrence of a given target value.


#include <iostream>
#include <vector>

int findFirstOccurrence(const std::vector<int>& arr, int target) {
    int low = 0, high = arr.size() - 1, result = -1;
    while (low <= high) {
        int mid1 = low + (high - low) / 3;
        int mid2 = high - (high - low) / 3;
        if (arr[mid1] == target) {
            result = mid1;
            high = mid1 - 1;
        } else if (arr[mid2] == target) {
            result = mid2;
            high = mid2 - 1;
        } else if (target < arr[mid1]) high = mid1 - 1;
        else if (target > arr[mid2]) low = mid2 + 1;
        else {
            low = mid1 + 1;
            high = mid2 - 1;
        }
    }
    return result;
}

int main() {
    std::vector<int> arr = {1, 2, 2, 2, 3, 4};
    int target = 2;
    int result = findFirstOccurrence(arr, target);
    if (result != -1) std::cout << "First occurrence at index " << result << std::endl;
    else std::cout << "Element not found" << std::endl;

1. Finding the Square Root of a Number

Given a non-negative integer, find its square root without using built-in functions.


#include <iostream>

double sqrtTernarySearch(double x) {
    double low = 0, high = x, mid1, mid2;
    while (high - low > 1e-6) {
        mid1 = low + (high - low) / 3;
        mid2 = high - (high - low) / 3;
        if (mid1 * mid1 > x) high = mid1;
        else if (mid2 * mid2 < x) low = mid2;
        else {
            low = mid1;
            high = mid2;
        }
    }
    return (low + high) / 2;
}

int main() {
    double x = 16;
    std::cout << "Square root of " << x << " is " << sqrtTernarySearch(x) << std::endl;
    return 0;
}

2. Finding the Peak Element in an Array

Given an array where each element is greater than or equal to its neighbors, find a peak element.


#include <iostream>
#include <vector>

int findPeakElement(const std::vector<int>& nums) {
    int low = 0, high = nums.size() - 1;
    while (low <= high) {
        int mid1 = low + (high - low) / 3;
        int mid2 = high - (high - low) / 3;
        if ((mid1 == 0 || nums[mid1 - 1] <= nums[mid1]) &&
            (mid1 == nums.size() - 1 || nums[mid1 + 1] <= nums[mid1]))
            return mid1;
        else if (mid1 > 0 && nums[mid1 - 1] > nums[mid1]) high = mid1 - 1;
        else low = mid1 + 1;
    }
    return -1;
}

int main() {
    std::vector<int> nums = {1, 3, 20, 4, 1};
    std::cout << "Peak element at index " << findPeakElement(nums) << std::endl;
    return 0;
}

Given a rotated sorted array, find the minimum element.


#include <iostream>
#include <vector>

int findMinInRotatedArray(const std::vector<int>& nums) {
    int low = 0, high = nums.size() - 1;
